# For You Accounting

An Accounting Firm Website Built With Gatsby.

## Software & Resources

- React
- Gatsby v3
- Gatsby Plugins
- Graphql query
- SCSS
- Bootstrap 4 grid 


## Develop Locally

**Prerequisites:** gatsby 3 and node-sass 5 requires node v12 or higher

```
npm install
```

```
npm run develop
```

```
npm run build
```
